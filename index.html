Cheatsheet for HTML and CSS

Tag Name Description
p Paragraph For long bouts of text
a Anchor Used for making something an anchor to a link
ol Ordered List Makes a numerical list
ul Unordered list Makes an unordered list
link Link Links stylesheets, fonts, images, etc.
src= url() URL Speficies a url to be linked
img Image Used to link an image
video Video Used to link a video
audio Audio Used to link audio
Attribute Name Description
href="" Relative Link Used in link tag to specify linking a file in the same directory folder
src="" Source Specifies that a url is about to be linked
controls controls Gives user controls of a video element. Pausing, skipping, audio, etc.
href= "#" Linking to the same page Use this if you want to place anchor points to skip around the page
class="" Class Assignes the element into a class, which can be used on multiple elements
id="" ID Most specific indicator, overrides rules, use sparingly.
